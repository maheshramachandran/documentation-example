Algebra

A set \(\mathcal{A}\) formed from another set \(S\), according 3 rules:

  • \(S \in \mathcal{A}\)

  • If \(X \in \mathcal{A}\), then \(X^c \in \mathcal{A}\)

  • If \(X \in \mathcal{A}\) and \(Y \in \mathcal{A}\), then \(X \cup Y \in \mathcal{A}\)

Implications of rules:

  • Empty set is always in \(\mathcal{A}\), because the empty set is the complement of \(S\).

  • The union of all pairs of sets in \(S\) must be in \(\mathcal{A}\).

  • The intersections of sets is included in \(\mathcal{A}\).

  • The relative differences of sets is also included in \(\mathcal{A}\).

Sigma Algebra

A sigma algebra is a type of algebra, the follows the first two rules of an algebra, but expands rule 3 require that the union of any sequence of sets in \(\mathcal{A}\) must also be in \(\mathcal{A}\):

  • If \(\{A_n: n \in N \}\) is a sequence of sets in \(\mathcal{A}\), then \(\bigcup_{n} A_n \in \mathcal{A}\).

Given a domain \(S\), and a set \(C\) of subsets of \(S\) that you want to measure, you generate a sigma algebra \(\mathcal{A} = \sigma(C)\).

For example, if \(C = { A, B }\), you form a sigma algebra \(\mathcal{A} = \sigma( A, B)\). You get the sigma algebra with the fewest elements that also contains the set \(C\).
